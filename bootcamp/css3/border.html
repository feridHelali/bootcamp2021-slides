Bordures CSS

Les propriétés de bordure CSS vous permettent de spécifier le style, la largeur et la couleur de la bordure d'un élément.

Bordures CSS

border-style
La border-style propriété spécifie le type de bordure à afficher. Les valeurs suivantes sont autorisés :
dotted - dashed - solid - double - groove - ridge - inset - outset - none - hidden

Bordures CSS

border-style
              
  p.dotted {border-style: dotted;}
  p.dashed {border-style: dashed;}
  p.solid {border-style: solid;}
  p.double {border-style: double;}
  p.groove {border-style: groove;}
  p.ridge {border-style: ridge;}
  p.inset {border-style: inset;}
  p.outset {border-style: outset;}
  p.none {border-style: none;}
  p.hidden {border-style: hidden;}
  p.mix {border-style: dotted dashed solid double;}

Bordures CSS

border-width
La border-width propriété spécifie la largeur des quatre bordures. 
              
  p.one {
    border-style: solid;
    border-width: 5px;
  }
  
  p.two {
    border-style: solid;
    border-width: medium;
  }
  
  p.three {
    border-style: dotted;
    border-width: 5px 20px; /* 5px top and bottom, 20px on the sides */
  }
  
  p.four {
    border-style: dotted;
    border-width: thick;
  
  p.five {
    border-style: solid;
    border-width: 25px 10px 4px 35px; /* 25px top, 10px right, 4px bottom and 35px left */
  }

Bordures CSS

border-width
La border-color propriété est utilisée pour définir la couleur des quatre bordures.
Comme on a vu précédemment, la couleur peut définie par : nom - code HEX - RVB - HSL 
              
  p.one {
    border-style: solid;
    border-color: red;
  }             
  p.two {
    border-style: solid;
    border-color: red green blue yellow; /* red top, green right, blue bottom and yellow left */
  }
  p.three {
    border-style: solid;
    border-color: #ff0000; /* red */
  }
  p.four {
    border-style: solid;
    border-color: rgb(255, 0, 0); /* red */
  }
  p.five {
    border-style: solid;
    border-color: hsl(0, 100%, 50%); /* red */
  }

Bordures CSS

border-width
En CSS, il existe des propriétés pour spécifier chacune des bordures (haut, droite, bas et gauche) : 
              
  p {
    border-top-style: dotted;
    border-right-style: solid;
    border-bottom-style: dotted;
    border-left-style: solid;
  }

Bordures CSS

border shorthand
La border propriété est une propriété abrégée pour les propriétés de bordure individuelles suivantes dans l'ordre:
border-width | border-style(obligatoire) | border-color 
              
  p {
    border: 5px solid red;
  }

Bordures CSS

Les bordures arrondies
La border-radius propriété est utilisée pour ajouter des bordures arrondies à un élément 
              
  p {
    border: 2px solid red;
    border-radius: 5px;
  }
