Alignement du texte
La text-align propriété est utilisée pour définir l'alignement horizontal d'un texte. Exemple: 
              
                h1 {
                  text-align: center;
                }              
                h2 {
                  text-align: left;
                }              
                h3 {
                  text-align: right;
                }
                div {
                  text-align: justify;
                }

Alignement du texte en dernier
              
                p.a {
                  text-align-last: right;
                }             
                p.b {
                  text-align-last: center;
                }              
                p.c {
                  text-align-last: justify;
                }

Décoration du texte
La text-decoration propriété est utilisée pour définir ou supprimer des décorations du texte.
              
                a {
                  text-decoration: none;
                }
                h2 {
                  text-decoration: overline;
                } 
                h3 {
                  text-decoration: line-through;
                }              
                h4 {
                  text-decoration: underline;
                }

Transformation du texte
La text-transform propriété est utilisée pour spécifier des lettres majuscules et minuscules dans un texte.
              
                p.uppercase {
                  text-transform: uppercase;
                }              
                p.lowercase {
                  text-transform: lowercase;
                } 
                p.capitalize {
                  text-transform: capitalize;
                }

Indentation du texte

                p {

Espacement des lettres
La letter-spacing propriété est utilisée pour spécifier l'espace entre les caractères dans un texte.
              
                h1 {
                  letter-spacing: 5px;
                }

Ombre du texte
La text-shadow propriété ajoute une ombre au texte.
Dans son utilisation la plus simple, vous ne spécifiez que l'ombre horizontale (2px) et l'ombre verticale (2px) : 
              
                h1 {
                  text-shadow: 2px 2px;
                }

Texte css

Ombre du texte
Ensuite, ajoutez une couleur (rouge) à l'ombre : 
              
                h1 {
                  text-shadow: 2px 2px red;
                }

Texte css

Ombre du texte
Ensuite, ajoutez un effet de flou (5px) à l'ombre : 
              
                h1 {
                  text-shadow: 2px 2px 5px red;
                }

Font-style
La font-style propriété est principalement utilisée pour spécifier du texte en italique.
Cette propriété a trois valeurs:
  • normal-Le texte s'affiche normalement
  • italique - Le texte est affiché en italique
  • oblique - Le texte est penché

Police css

Font-style
              
      p.normal {
       font-style: normal;
      }
                
      p.italic {
        font-style: italic;
      }
                
      p.oblique {
       font-style: oblique;
     }

Font-Weight

Font-variant

      p.thick {

Font-size
La font-size propriété définit la taille du texte.
On peut définir la taille du police en Pixels
              
      h1 {
        font-size: 40px;
      }
                
      h2 {
        font-size: 30px;
      }
                
      p {
        font-size: 14px;
      }

Police css

Font-size
On peut définir la taille du police avec des Em
pixels / 16 = em 
              
      h1 {
        font-size: 2.5em; /* 40px/16=2.5em */
      }
                
      h2 {
        font-size: 1.875em; /* 30px/16=1.875em */
      }
                
      p {
        font-size: 0.875em; /* 14px/16=0.875em */
      }

Police css

Font-size
La taille du texte peut être définie avec une vw (Viewport) unité, ce qui signifie la "largeur de la fenêtre". De cette façon, la taille du texte suivra la taille de la fenêtre du navigateur : 
              
                <h1 style="font-size:10vw">Hello World</h1>
Google Font
Si vous ne souhaitez utiliser aucune des polices standard en HTML, vous pouvez utiliser les polices Google.
Il faut ajouter un lien spécial dans le Header, puis reportez-vous à la police dans le CSS. 
              
                
  <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Sofia">
  <style>
   body {
     font-family: "Sofia", sans-serif;
   }
  </style>
