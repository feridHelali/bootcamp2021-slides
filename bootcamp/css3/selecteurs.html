Les Sélecteurs

Les Sélecteurs CSS sont utilisés pour identifier les éléments HTML qu'on souhaite styliser.
Ils sont diviser en 5 catégories:
  1. Sélecteurs simples (sélectionnez les éléments en fonction du nom d'élément, de l'identifiant, de la classe)
  2. Sélecteurs de combinateurs (sélectionnez des éléments en fonction d'une relation spécifique entre eux)
  3. Sélecteurs de pseudo-classe (sélectionnez des éléments en fonction d'un certain état)
  4. Sélecteurs de pseudo-éléments (sélectionnez et stylisez une partie d'un élément)
  5. Sélecteurs d'attributs (sélectionnez des éléments en fonction d'un attribut ou d'une valeur d'attribut)

Les sélecteurs simples

Les sélecteurs d'éléments
Le sélecteur d'élément sélectionne les éléments HTML en fonction du nom de l'élément.

      p {
        text-align: center;
        color: red;
      }
Ici, tous les éléments <p> de la page seront alignés au centre, avec une couleur de texte rouge.

Les sélecteurs simples

Les sélecteurs d'identifiant
Le sélecteur id utilise l'attribut id d'un élément HTML pour sélectionner un élément spécifique.
      
        #para1 {
          text-align: center;
          color: red;
        }
La règle CSS ci-dessus sera appliquée à l'élément HTML avec id="para1".

Les sélecteurs simples

Les sélecteurs de classe
Le sélecteur de classe sélectionne des éléments HTML avec un attribut de classe spécifique. 
      
        .center {
          text-align: center;
          color: red;
        }
Dans cet exemple, tous les éléments HTML avec class="center" seront rouges et alignés au centre

Les sélecteurs combinateurs

Un sélecteur CSS peut contenir plusieurs sélecteurs simples. Entre les sélecteurs simples, nous pouvons inclure un combinateur.
Il existe quatre combinateurs différents en CSS :
  • sélecteur descendant (espace)
  • sélecteur enfant (>)
  • sélecteur de frère adjacent (+)
  • sélecteur général de frères et sœurs (~)

Les sélecteurs de pseudo-classe

Une pseudo-classe est utilisée pour définir un état particulier d'un élément.
Par exemple, il peut être utilisé pour :
  • Styliser un élément lorsqu'un utilisateur passe la souris dessus
  • Styler différemment les liens visités et non visités
  • Styler un élément lorsqu'il obtient le focus

La syntaxe

      
        selector:pseudo-class {
          property: value;
        }

Exemple

Les liens peuvent être affichés de différentes manières 
      
        /* unvisited link */
        a:link {
         color: #FF0000;
        }

        /* visited link */
        a:visited {
         color: #00FF00;
        }

        /* mouse over link */
        a:hover {
          color: #FF00FF;
        }

        /* selected link */
        a:active {
          color: #0000FF;
        }

Les sélecteurs pseudo-éléments

Un pseudo-élément CSS est utilisé pour styliser des parties spécifiées d'un élément.
Par exemple, il peut être utilisé pour :
  • Styliser la première lettre, ou ligne, d'un élément
  • Insérer du contenu avant ou après le contenu d'un élément

La syntaxe

      
        selector::pseudo-element {
          property: value;
        }
Le ::first-line pseudo-élément est utilisé pour ajouter un style particulier à la première ligne d'un texte.
L'exemple suivant formate la première ligne du texte dans tous les éléments

: 

        
          p::first-line {
            color: #ff0000;
            font-variant: small-caps;
          }
Le ::first-letter pseudo-élément est utilisé pour ajouter un style particulier à la première lettre d'un texte.
L'exemple suivant formate la première lettre du texte dans tous les éléments

: 

        
          p::first-letter {
            color: #ff0000;
            font-size: xx-large;
          }

Les sélecteurs d'attributs

Il est possible de styliser les éléments HTML qui ont des attributs ou des valeurs d'attribut spécifiques.

L'exemple suivant sélectionne tous les éléments <a> avec un attribut target="_blank" 
        
          a[target="_blank"] {
            background-color: yellow;
          }

La valeur !important

Le mot !important peut être ajouté à n'importe quelle valeur dans une règle CSS. Il rend la règle prioritaire.

Les priorités des sélecteurs

Tous les sélecteurs n'ont pas la même priorité. En cas de contradiction entre deux règles, celle qui s'applique est celle qui a le sélecteur le plus prioritaire. De façon générale, les sélecteurs ont une priorité d'autant plus grande qu'ils sont plus précis. Le sélecteur très général * a une priorité de 0. A l'inverse, un sélecteur sur un identifiant , qui en principe ne concerne qu'un seul élément de la page, a une priorité de 100.

Un commentaire CSS est placé à l'intérieur de l' <style> élément. 
      
        /* This is a single-line comment */
        p {
          color: red;
        }

Remarque

L'identation est très importante pour la clarté du code.
